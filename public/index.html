Cuny Books Scraper

Version 1.0

About?

Cuny Book Scraper is designed to get information of a Book through Cuny Catalog. By giving us a doc number of your book of choice, we will return you the details of that book in a json format. Be free to checkout the Usage.

Usage

Cuny Book Scraper is built in with simplicity and conviniency:

  • Provide a Doc Number
    For example, Doc Number = 007576009
    http://cunybooksscraper.heroku.com/docnum/007576009
  • JSON
